To configure this plugin, you need to download Kiuwan local analyzer and configure it. Follow this steps:
  • 1. Create a Kiuwan account.
  • 2. Login and create your first Kiuwan application.
  • 3. Download local analyzer and unzip it in any folder.
  • 4. Run FOLDER/bin/agentGUI.cmd and configure your user and password. You can launch your first analysis in order to know if everything is correct.
